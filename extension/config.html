Extension Configuration

Open OBS, go to the menu on the top and click on Docks>Custom Browser Docks:
Fill in a name you will recognize for the dock, and paste the URL:
https://stg.chronx.dev/chat-plays-obs/index.html#/
hit "Apply" and a new window dock with the name you selected should pop up.
Follow the instructions provided in the newly created Dock.
Add the dock to a part of your obs if you want to be able to change things live.


If you would like to run this locally you can pull the project from streamerTwitchJam github project
